# Data Talk 📊🤖

**Data Talk** is an intelligent, chat-based platform that allows you to analyze, clean, and visualize your datasets using the power of Google's Gemini AI. Simply upload an Excel or CSV file, and use natural language to ask questions, generate charts, and get insights about your data!

---

## ✨ Features

- **Conversational Data Analysis:** Chat directly with your data using Gemini.
- **Automated Data Cleaning:** The system handles messy data, standardizing columns and types behind the scenes.
- **Instant Visualizations:** Ask for a bar chart or a scatter plot, and the AI will write and execute the code to generate the image instantly in the chat.
- **Data Viewer:** See a robust, interactive grid view of the dataset you are currently querying.
- **Secure Code Execution:** Python code generated by the LLM is executed in a controlled backend environment.

---

## 🛠️ Technology Stack

- **Backend:** Python, Flask server, Pandas (for data manipulation)
- **AI Integration:** Google GenAI SDK (`google-genai`), targeting the Gemini models
- **Frontend:** HTML, Vanilla JavaScript, CSS
- **Data Formats:** `.csv`, `.xlsx`

## 🚀 Getting Started

### 1. Prerequisites
- **Python 3.10+**
- A **Google Gemini API Key**. You can get one from Google AI Studio.

### 2. Setup your Environment

Rename or create your `.env` file in the root directory:
```env
# Create a .env file locally and add your API key:
GEMINI_API_KEY="your-api-key-here"

# Optional settings
FLASK_DEBUG=False
PORT=5000
MAX_UPLOAD_MB=10
```

### 3. Running the App (Windows)

Just double-click the `start.bat` file! 

This will automatically:
1. Check if Python is installed.
2. Install the necessary dependencies from `requirements.txt`.
3. Start the Flask backend server.
4. Launch the web application in your default browser.

**Manual Start:**
If you prefer terminal commands:
```bash
# 1. Create a virtual environment (optional but recommended)
python -m venv .venv
.venv\Scripts\activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Start the server
python server.py
```
Then navigate to `http://localhost:5000` in your web browser.

## 📂 Project Structure

- `server.py` - Standard Flask routing and REST API endpoints.
- `data_service.py` - Core logic for loading, validating, and cleaning the user's uploaded Pandas Dataframes.
- `gemini_service.py` - Handles communication with the Gemini AI model, structuring the system prompts.
- `code_executor.py` - Responsible for securely running Python code generated by the LLM to process data and create charts.
- `dashboard.html` / `script.js` / `styles.css` - The frontend UI containing the chat interface and data table.
- `uploads/` - Temporary storage for datasets during runtime (ignored by version control).
- `requirements.txt` - Python dependencies needed.

---

## 🔒 Note on Security

Be mindful that this application uses a code executor (`code_executor.py`) to run AI-generated Python scripts on your local machine. Never expose this app to the public internet without implementing robust sandboxing (e.g., Docker containers) and stronger security constraints. It is currently designed as a helpful local tool.
